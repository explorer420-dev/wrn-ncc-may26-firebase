Test Mode

To simulate a specific event day, set test_day_override in Firestore → config → event to a number 1–6. Set to 0 to use the real date. The topbar shows 🧪 TEST when active.

Exporting Data

Use ⬇ Download CSV in either tab. The Registered export respects the current search filter. The Log export includes all entries including test-mode rows — filter by the TEST column in your spreadsheet if needed.

Troubleshooting
Dashboard data is staleRe-run the morning import from Google Sheets
Wrong day shownCheck test_day_override in Firestore config — set to 0 for live date
Attendance log not updatingRefresh the page; check internet connection
Signed in as wrong userClick Sign out (top right) and re-login with correct credentials
